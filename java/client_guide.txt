ИНСТРУКЦИЯ ПО РАБОТЕ И СБОРКЕ КЛИЕНТА

Описание:
Реализация клиента представляет собой консольное приложение для работы с SFTP-сервером. 
Для подключения к серверу требуется настроить ./resources/config.properties. Это можно сделать как при запуске приложения, так и в текстовом редакторе самого файла.

Сборка выполняется из корневой папки проекта командой: mvn clean package
ПРИМЕЧАНИЕ: Для сборки приложения требуется JDK версии 8 и выше, Maven

Запуск выполняется командой: java -jar ./*путь к файлу*/infotecs-1.0.jar
ПРИМЕЧАНИЕ: Для запуска приложения требуется JRK версии 8 и выше

Начало работы:
При запуске приложения появляется меню настройки файла конфигурации клиента.

=====================SFTP CLIENT=====================
Do you want to change these parameters? (y/n)
1. Host: *IP хоста*
2. Port: *порт хоста*
3. User: *имя пользователя для подключения*
4. Password: *пароль*
5. Local directory: *директория для загрузки файла с целового хоста*
6. File path: *путь к файлу на хосте*
>

Параметр Local directory определяется автоматически при запуске приложения, но его можно поменять в рамках текущей сессии.

При изменении параметров конфига предлагается выбрать нужный параметр по его номеру или поменять все сразу.

====================CONFIG EDITOR====================
Select what do you want to change:
1. Host: *IP хоста*
2. Port: *порт хоста*
3. User: *имя пользователя для подключения*
4. Password: *пароль*
5. Local directory: *директория для загрузки файла с целового хоста*
6. File path: *путь к файлу на хосте*
7. All.
0. Save and exit.
>

После подтверждения параметров конфига, клиент пробует подключиться к SFTP-серверу. В случае удачного подключения выводится функциональное меню.

==================HOST *host:port* ==================
Menu:
1. Print domain-ip pair list.
2. Get IP by domain.
3. Get domain by IP.
4. Add new domain-ip pair.
5. Delete domain-ip pair.
0. Exit.
>

Дальнейшая работа с приложением идет в соответствии с выводимыми на экран инструкциями (ввод IP-адреса, домена, пары домен-IP).
